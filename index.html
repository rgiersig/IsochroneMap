IsochroneMap

Draw an isochrone city map overlay using data from a public transport authority

Download this project as a .zip file Download this project as a tar.gz file

Using station and connection data from a public transport authority an overlay using the google maps api is created that shows how long it takes from a certain point in the city to any other point, either by walking or by public transport.

Prototype for Vienna
